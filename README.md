# 🎾 Padel MVP — Telegram-бот + CRM для записи на игры

Это твой первый большой проект! Здесь два приложения:

1. **`bot.py`** — Telegram-бот, с которым общаются игроки: заполняют анкету, смотрят игры, записываются.
2. **`app.py`** — веб-сайт (CRM) для администратора: создание игр, просмотр заявок, подтверждение оплат.

Оба приложения используют одну и ту же базу данных PostgreSQL (в Neon).

Инструкция написана специально для **MacBook** и рассчитана на то, что ты вообще ничего
из этого раньше не делал. Иди по шагам подряд, ничего не пропускай.

---

## Шаг 0. Что уже есть, а что нужно получить самому

Тебе понадобится **токен Telegram-бота**. Вот как его получить:

1. Открой Telegram, найди бота `@BotFather`.
2. Напиши ему `/newbot`.
3. Придумай имя бота (например, "Padel Booking Bot") и username (должен заканчиваться на `bot`, например `my_padel_bot`).
4. BotFather пришлёт тебе токен — длинную строку вида `123456789:AAExampleTokenHere`. Сохрани её, она понадобится в шаге 4.

Строка подключения к базе данных PostgreSQL (Neon) у тебя уже есть, она указана в файле `.env`.

⚠️ **Важно:** пароль от базы данных ты присылал в открытом чате. Рекомендуется зайти в
[консоль Neon](https://console.neon.tech) → Settings → Reset password и сгенерировать новый пароль,
а потом обновить его в файле `.env`. Никому не показывай содержимое `.env`.

## Шаг 1. Установка Python на MacBook

Проверь, установлен ли у тебя Python 3.11 или новее. Открой приложение **Терминал**
(искать через Spotlight — `Cmd + Space`, ввести "Terminal") и напиши:

```bash
python3 --version
```

- Если видишь что-то вроде `Python 3.11.5` или новее — Python уже есть, переходи к Шагу 2.
- Если видишь ошибку или версия ниже 3.10, установи Python:
  1. Установи [Homebrew](https://brew.sh) (менеджер пакетов для Mac), если его ещё нет:
     ```bash
     /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
     ```
  2. Установи Python:
     ```bash
     brew install python@3.12
     ```
  3. Проверь снова: `python3 --version`

## Шаг 3. Создай виртуальное окружение

Виртуальное окружение — это "изолированная коробка" с зависимостями именно для этого
проекта, чтобы они не мешали другим твоим проектам на компьютере.

```bash
python3 -m venv venv
```

Активируй его:

```bash
source venv/bin/activate
```

После этого в начале строки терминала появится `(venv)` — это значит, что окружение активно.
**Каждый раз, когда открываешь новый терминал для этого проекта, нужно заново выполнять
эту команду активации.**

## Шаг 5. Настрой файл .env

В папке проекта уже есть файл `.env` с заготовкой. Открой его любым текстовым редактором
(например, TextEdit или VS Code) и заполни:

```
BOT_TOKEN=вставь_сюда_токен_от_BotFather
DATABASE_URL=postgresql://neondb_owner:...  (уже заполнено, но проверь)
ADMIN_LOGIN=admin
ADMIN_PASSWORD=придумай_свой_пароль
FLASK_SECRET_KEY=любая_случайная_строка
```

Чтобы сгенерировать случайную строку для `FLASK_SECRET_KEY`, можно выполнить в терминале:

```bash
python3 -c "import secrets; print(secrets.token_hex(32))"
```

и вставить результат в `.env`.

## Шаг 7. Запусти бота и CRM одновременно

Тебе нужно **два открытых окна терминала одновременно** (бот и сайт — это два разных
процесса, которые должны работать параллельно).

**Окно терминала №1 — бот:**

```bash
cd ~/Documents/padel_mvp
source venv/bin/activate
python bot.py
```

Увидишь `Бот запущен...` — значит, всё работает. Найди своего бота в Telegram
(по username, который придумал в BotFather) и напиши `/start`.

**Окно терминала №2 — CRM (открой новую вкладку терминала, `Cmd + T`):**

```bash
cd ~/Documents/padel_mvp
source venv/bin/activate
python app.py
```

Увидишь адрес вида `http://127.0.0.1:5000`. Открой его в браузере.
Войди, используя `ADMIN_LOGIN` и `ADMIN_PASSWORD`, которые ты указал в `.env`.

## Как пользоваться

### Как админ (через CRM в браузере):
1. Зайди на `http://127.0.0.1:5000`, залогинься.
2. На странице **"Игры"** нажми **"+ Создать игру"** — укажи дату, время, место, цену, количество мест.
3. Когда игроки запишутся, на странице **"Заявки"** ты увидишь их — измени статус на "подтверждена".
4. После подтверждения заявки на странице **"Оплаты"** появится ожидающая оплата — нажми "Подтвердить оплату", когда деньги получены.
5. На странице **"Игры"** есть кнопка **"Выгрузить отчёт в Excel"** — скачает таблицу со всеми играми, числом записавшихся и собранными деньгами.

### Как игрок (через Telegram-бота):
1. Написать `/start` — бот попросит заполнить анкету (имя, телефон, уровень).
2. `/games` — покажет список игр с кнопками "Записаться".
3. `/my_bookings` — покажет твои записи с кнопкой "Отменить запись".
4. За 24 часа до игры бот сам пришлёт напоминание.

## Частые проблемы

**"ModuleNotFoundError: No module named 'aiogram'"**
→ Ты не активировал виртуальное окружение. Выполни `source venv/bin/activate` и снова `pip install -r requirements.txt`.

**Бот не отвечает в Telegram**
→ Проверь, что в терминале с `python bot.py` нет ошибок и что `BOT_TOKEN` в `.env` правильный (без лишних пробелов).

**Ошибка подключения к базе данных**
→ Проверь `DATABASE_URL` в `.env`. Убедись, что у тебя есть интернет — Neon это облачная база данных.

**Порт 5000 уже занят**
→ На новых Mac иногда порт 5000 занят системным сервисом AirPlay. Отключи его:
Системные настройки → Основные → AirDrop и Handoff → выключи "AirPlay Receiver". Либо в `app.py`
измени `app.run(debug=True, port=5000)` на другой порт, например `port=5050`.
